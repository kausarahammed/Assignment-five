What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a plotting interface, or structure, that represents HTML and XML documents. Documer exposes all elements through a structure, where it is viewed as an object. These objects can be written and modified by plotting languages ​​such as JavaScript.

How do you select an element from the DOM?

select elements specified Id :

let element = document.getElementById("myElementId");

To select multiple elements specific class:

let element = document.getElementByClassName("myClassName");

To select elements by specified tag name:

let elements = document.getElementsByTagName("p");

To select an element like a CSS selector:

let element = document.querySelector(".myClassName");

To select multiple elements like a CSS selector:

let element = document.querySelectorAll(".myClassName");

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where events are attached to their parent elements (parent elements) instead of specific smaller elements (elements). This causes the event to bubble up to that parent content, and event handlers work for many objects.

How do you manipulate an element's attributes and styles using the DOM?

let element = document.getElementById("myElement");

let value = element.getAttribute("data-info");

element.setAttribute("data-info");

element.removeAttribute("data-info");
